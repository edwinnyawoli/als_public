CountryChartData

  • country : String
    Alpha-2-code of country
  • count : Integer
    The total number of libraries recorded for this country in the system. This includes libraries of all types and statuses.
  • estimatedTotal : Integer
    The estimated total number of libraries in the country as provided by an Administrator
  • typeData : List
    An array of library counts by type belonging to this country
  • statusData : List
    An array of library counts by status belonging to this country
