This is the API

getLibraryChart

Get chart data on libraries

URL

http://localhost:3000/api/charts/libraries

HTTP Method

GET

Response Type

LibraryChart

Parameters

  • Query: countries List

    Array of countries to filter by. Countries are specified using their alpha-2-code. Absence represents all countries.

  • Query: otherLibraryType String

    Custom library type when OTHER is specified as type

  • Query: type String

    Type of library

  • Query: status String

    Status of library
