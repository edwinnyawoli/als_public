This is the API

getLibraries

Get libraries

URL

http://localhost:3000/api/libraries

HTTP Method

GET

Response Type

List

Parameters

  • Query: type String

    Type of library

  • Query: format String

    The format of the response body

getLibraryImages

Get images of library. Returns an array or urls of these images

URL

http://localhost:3000/api/libraries/{libraryId}/images

HTTP Method

GET

Response Type

List

Parameters

  • Path: libraryId String

getLibraryLogs

Get activity logs for a library

URL

http://localhost:3000/api/libraries/{libraryId}/logs

HTTP Method

GET

Response Type

List

Parameters

  • Path: libraryId String
